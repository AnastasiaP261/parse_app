Обязательные параметры запроса:

url - адрес страницы, данные которой будут парситься. Адрес должен быть валидным и существующим.


Необязательные параметры запроса:

output_text_width - ширина выходного файла в символах. Значение должно быть целым числом больше 0. По умолчанию - 110. Не стоит выставлять это значение слишком малым. В таком случае многие слова могут быть порезаны на несколько строк

saving_img_links - принимает значения True/true/1 или False/false/0. При True - будут сохранены ссылки на изображения, если они были в контенте страницы. При False ссылки сохранены не будут. По умолчанию True.

file_name - позволяет задать имя для выходного файла. Имя может содержать только латинские буквы в верхнем и нижнем регистре, цифры, а также знак тире и нижнего подчеркивания. По умолчанию answer. УКАЗЫВАЬ РАСШИРЕНИЕ ФАЙЛА НЕ ТРЕБУЕТСЯ.


Пример корректного запроса:

http://127.0.0.1:8000/parse/?url=https://en.wikipedia.org/wiki/Python_(programming_language)&output_text_width=200
